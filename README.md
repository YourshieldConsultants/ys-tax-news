# ys-tax-news — публичная лента новостей YS Tax Calculator

Только ОПУБЛИКОВАННЫЙ контент: `news.json` (контракт — `schema/news.schema.json`) и `images/`.
Публикует ТОЛЬКО редакционный робот после апрува Owner в Telegram (архитектура — D-059 в
репозитории приложения). Правки руками в обход робота ломают content_hash и будут отвергнуты
валидацией CI. Никакого кода, промптов и черновиков здесь нет — они в приватном ys-tax-news-bot.
